Zomato Mumbai Restaurant Analysis & Prediction
This project performs a comprehensive analysis of Zomato restaurant data in Mumbai. It includes data cleaning, exploratory data analysis (EDA) to visualize market trends, and Machine Learning models to predict restaurant "REVIEWS" by Using Various Machine Learning Models Like 'Logistic Regression', 'K-Nearest Neighbors', 'Decision Tree', 'Random Forest', 'XGBoost'

Zomato_Mumbai_Dataset.csv: The raw, unprocessed dataset containing Almost [15,000] rows and columns like Restaurant Name, Address, Cuisines, Price , time , Review , votes, etc. Used in "Zomato Mumbai Analysis" Project
Final_Clean_Zomato_Mumbai_Dataset.csv: The cleaned version of the dataset used for Machine Learning Model Building. Used in "Zomato Mumbai ML Models"
